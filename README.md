# Physics1 Example

Este proyecto es un ejemplo básico de física desarrollado en C para el canal [Make Classic Games](https://www.twitch.tv/makeclassicgames) en Twitch. El objetivo es mostrar conceptos fundamentales de simulación física aplicados a videojuegos clásicos.

## Características

- Simulación sencilla de física 2D en C.
- Código comentado y fácil de entender.
- Ideal para aprender y experimentar con motores físicos básicos.

## Requisitos

- Compilador C (gcc, clang, etc.)
- Dependencia de la librería Raylib. Puedes encontrar el repositorio en [github.com/raysan5/raylib](https://github.com/raysan5/raylib).

## Uso

1. Clona el repositorio.
2. Compila el proyecto usando tu compilador C preferido (usa ```make``` para compilar).
3. Ejecuta el ejemplo siguiendo las instrucciones del archivo principal.
